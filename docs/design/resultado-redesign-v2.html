Reveal parcial: so o nome + 3 comportamentos. O corpo longo de 200 palavras foi removido. Nao fecha o loop emocional.
Mudanca #3
Transicao forte: nude → burgundy dark na secao de perguntas. Marca visualmente "aqui comeca a oferta". Referencia: Noom, Hims.

Blocos bloqueados com blur: 2 visiveis + 2 borrados com cadeado. Mostra o que ela perde se nao comprar. Referencia: 16Personalities.

Mudanca #6
