# PRD — UNLMTD Brandbook Completo

**Projeto:** UNLMTD Brand Identity System
**Epic:** 5 — UNLMTD Brandbook
**Data:** 2026-03-27
**Status:** Aprovado para execução
**Owner:** David Levy / Carlos Pimenta

---

## 1. Contexto & Problema

A UNLMTD é uma agência de criação e estratégia que trabalhou com Unilever, Google, Coca-Cola, Heineken, Itaú, Samsung e outros grandes players. A marca possui identidade visual existente (logo wordmark + símbolo ∞ + paleta black/cream), mas **não tem um brandbook sistematizado** que oriente produções internas e externas de forma consistente.

Sem esse sistema formalizado, cada produção reinventa a roda — especialmente em formatos digitais (Instagram, apresentações, propostas) — diluindo a força da marca.

---

## 2. Objetivo

Criar um **Brandbook completo e aplicável** da UNLMTD que:
- Documente e sistematize a identidade já existente
- Defina regras claras para todas as aplicações
- Entregue templates prontos para uso imediato
- Preserve a essência: fluida, plural, anti-fórmula

---

## 3. Manifesto da Marca (fonte de verdade)

> UNLMTD não é singular. É plural.
> UNLMTD não tem forma, formato ou fórmula. É experimentar.
> UNLMTD não é fronteira. É portas abertas.
> UNLMTD não é pensar fora da caixa. É não ter caixa, nem rótulos.
> UNLMTD é cocriação. É abraço. É ziriguigum e mojo.

## 6. Escopo de Entregáveis

### 6.1 Identidade de Marca
- [ ] Logo — variações completas (principal, negativo, positivo, monocromático, reduzido)
- [ ] Símbolo ∞ — variações e regras de uso
- [ ] Área de proteção e tamanho mínimo
- [ ] Usos incorretos (brand don'ts)
- [ ] Versões para sub-marcas / variações UNLMTD

### 6.2 Sistema de Cores
- [ ] Paleta primária: Preto + Cream (HEX, RGB, CMYK, Pantone)
- [ ] Paleta secundária: definir 3-4 cores de suporte
- [ ] Regras de combinação e hierarquia

### 6.3 Tipografia
- [ ] Família primária (display/headlines)
- [ ] Família secundária (corpo de texto)
- [ ] Escala tipográfica (H1→H6, body, caption)
- [ ] Regras de uso: peso, espaçamento, alinhamento

### 6.4 Iconografia
- [ ] Set de ícones no estilo da marca (line, minimal)
- [ ] Regras de uso e grid base

### 6.5 Tom de Voz
- [ ] Princípios de comunicação (baseados no manifesto)
- [ ] Como a UNLMTD fala (e como NÃO fala)
- [ ] Exemplos de copy: social, apresentação, proposta
- [ ] Guia de linguagem: PT-BR informal, direto, sem ego

### 6.6 Templates Instagram
- [ ] Formato Feed 4:5 (1080×1350px) — 3 variações
- [ ] Formato Stories 9:16 (1080×1920px) — 3 variações
- [ ] Grid visual coerente com a marca

### 6.7 Template de Apresentação
- [ ] Base em 16:9 (1920×1080px)
- [ ] Slides: capa, índice, título de seção, conteúdo texto, conteúdo visual, encerramento
- [ ] Adaptação do estilo das apresentações existentes (cream + black, tipografia bold)

### 6.8 Template de Proposta Comercial
- [ ] Base em A4 / formato digital
- [ ] Sections: capa, sobre a UNLMTD, o desafio, nossa proposta, equipe, investimento, próximos passos
- [ ] Coerente com as apresentações existentes

---

## 7. Critérios de Aceitação

- [ ] Brandbook entregue em PDF navegável
- [ ] Todos os assets em formatos editáveis (Figma ou equivalente)
- [ ] Templates prontos para uso imediato
- [ ] Consistência 100% entre todos os entregáveis
- [ ] Tom de voz validado com o manifesto
- [ ] Paleta secundária complementar à existente (não conflitante)

---

## 8. Restrições

- Preservar o wordmark e símbolo existentes (não redesenhar)
- Manter paleta black/cream como core
- Tom de voz: PT-BR, informal mas profissional, sem travessão
- Sem "special ones" — sistema para todo mundo da equipe usar

---

## 9. Público-Alvo do Brandbook

1. Equipe interna UNLMTD (uso nos projetos)
2. Parceiros e freelancers (produção de materiais)
