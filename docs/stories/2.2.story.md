# Story 2.2: Quiz Engine — 20 Perguntas + Scoring

## Status

InProgress

## Story

**As a** visitante que chegou pela Landing Page,
**I want** responder 20 perguntas sobre meus relacionamentos,
**so that** descubra qual é o meu estilo de apego.

## Acceptance Criteria

1. Tela de entrada ("Antes de começar") com botão para iniciar
2. 20 perguntas carregadas de `src/data/questions.js` (Versão B, 4 blocos)
3. Barra de progresso: texto "X de 20" + barra percentual animada (300ms)
4. Seleção de resposta avança automaticamente após 400ms (sem botão "próxima")
5. Animação fade entre perguntas (250ms)
6. Estado do quiz salvo em `localStorage` (tolerância a reload)
7. Tela de transição pós-quiz: "Seu resultado está sendo calculado." (4s)
8. Algoritmo de scoring: contagem A/B/C/D → estilo majoritário → navega para `/resultado/:estilo`
9. Estilos: `ansioso`, `distante`, `seguro`, `desorganizado` (ex-confuso)

## Dev Notes

### Scoring
- Cada resposta mapeia: A=ansioso, B=distante, C=seguro, D=desorganizado
- Contar ocorrências de cada estilo
- Empate: ordem de prioridade desorganizado > ansioso > distante > seguro

### Persistência
- `localStorage` key: `mjda_quiz_state`
- Salvar: `{ currentIndex, answers, started }`
- Limpar ao concluir (navegar para resultado)

### Animação fade
- CSS: opacity 0 → 1, 250ms ease-in-out
- Trigger: mudança de pergunta
