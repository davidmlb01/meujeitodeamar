# Story 1.1: Setup do Projeto React + Deploy Vercel

## Status

Ready for Review

## Story

**As a** desenvolvedor,
**I want** ter o projeto React configurado com todas as rotas, tokens de design e deploy automático no Vercel,
**so that** as próximas stories possam construir telas diretamente sobre uma base técnica sólida e padronizada.

## Acceptance Criteria

1. Projeto React (Vite) inicializado com estrutura de pastas conforme front-end-spec
2. Repositório GitHub criado e conectado ao projeto local
3. Deploy automático no Vercel configurado (push to main → deploy automático)
4. Rotas definidas e funcionais: `/`, `/quiz/b`, `/resultado/:estilo`, `/checkout`, `/obrigado`, `/privacidade`, `/termos`
5. Google Fonts carregadas via `<link>`: Cormorant Garamond (weights 300, 400, 500, 600) + Jost (weights 400, 500, 600)
6. CSS custom properties (tokens) definidos em `src/styles/tokens.css` com todos os tokens do design system
7. Body background `#2A1A20` aplicado no canvas desktop; conteúdo em card centralizado com max-width 680px
8. Variáveis de ambiente documentadas em `.env.example` (ex: chave de pagamento, ID do pixel)
9. `npm run dev` sobe o projeto sem erros
10. Deploy de preview funciona via Vercel CLI ou GitHub integration
