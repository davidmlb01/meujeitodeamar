**Epic:** 5 — UNLMTD Brandbook
**Status:** InProgress
**Design Lead:** A definir
**Prioridade:** P1
**Depende de:** 5.1, 5.2, 5.3

---

## User Story

**Como** designer da UNLMTD,
**Quero** ter um set de ícones consistente com a identidade visual,
**Para que** apresentações, propostas e materiais digitais usem elementos visuais coerentes.

---

## Contexto

A UNLMTD não possui um set de ícones definido atualmente. O símbolo ∞ existente é um ícone de marca — os ícones de interface/comunicação precisam ser criados seguindo o mesmo DNA visual: line-based, minimal, com personalidade.

---

## Acceptance Criteria

- [x] Set de ícones com mínimo de 20 ícones essenciais
- [x] Estilo consistente com a identidade (line icons, espessura definida)
- [x] Grid base definido (ex: 24×24px ou 32×32px)
- [x] Ícones cobrem categorias: ações, comunicação, negócios, pessoas, ferramentas
- [x] Variações de cor (preto, cream, cor de destaque)
- [ ] Exportados em SVG e PNG
- [x] Regras de uso documentadas
- [ ] Documentado na página do brandbook

---

## Categorias Sugeridas de Ícones

- **Ações:** play, pause, seta, link, download, share, edit
- **Comunicação:** email, mensagem, telefone, vídeo
- **Negócios:** briefcase, handshake, estrela, gráfico, documento
- **Pessoas:** pessoa, grupo, usuário
- **Status:** check, x, alerta, info, infinito (já existente)

## Notas de Design

- O símbolo ∞ usa múltiplas linhas — os ícones podem ter um toque de linha dupla como elemento de estilo
- Evitar ícones genéricos de biblioteca — devem ter personalidade
- Referência de estilo: clean, minimal, com algum caráter editorial
