# Story 3.1: Checkout — Pré-checkout + Kiwify Popup

## Status

Done

## Story

**As a** usuária que clicou em "Quero minha leitura completa por R$37",
**I want** ver um resumo do que estou comprando e as ofertas adicionais antes de pagar,
**so that** chegue ao pagamento com convicção e sem dúvidas.

## Acceptance Criteria

1. Cabeçalho: "Você está a um passo de entender o seu jeito de amar."
2. Resumo do pedido com 5 bullets + valor R$37
3. Order Bump 1: checkbox opt-in "Guia de Relacionamentos por Estilo" (+R$27)
4. Order Bump 2: checkbox opt-in "Leitura de quem eu amo" (+R$19)
5. Total dinâmico: R$37 + bumps selecionados
6. Botão CTA "Finalizar Pedido e Receber Minha Leitura" abre popup Kiwify
7. Popup Kiwify via embed script (`VITE_KIWIFY_PRODUCT_ID`)
8. Garantia: "Garantia incondicional de 7 dias"
9. Selos de segurança no rodapé
10. Fallback: se `VITE_KIWIFY_PRODUCT_ID` não configurado, redireciona para `VITE_PAYMENT_URL`

## Dev Notes

### Kiwify Popup API
```js
// Abre o popup nativo da Kiwify
window.KiwifyCheckout.open(productId)
```

### Script embed (index.html)
```html
<script src="https://widget.kiwify.com.br/checkout.js" defer></script>
```

### Lógica do botão
```js
const productId = import.meta.env.VITE_KIWIFY_PRODUCT_ID
if (productId && window.KiwifyCheckout) {
  window.KiwifyCheckout.open(productId)
} else {
  window.open(import.meta.env.VITE_PAYMENT_URL, '_blank')
}
```

### Order bumps
- OB1: +R$27 → `VITE_KIWIFY_OB1_ID` (produto separado ou order bump configurado na Kiwify)
- OB2: +R$19 → `VITE_KIWIFY_OB2_ID`
- Por ora: checkboxes afetam o total visual; os IDs reais serão configurados ao criar os produtos na Kiwify
