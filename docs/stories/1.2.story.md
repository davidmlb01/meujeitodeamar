# Story 1.2: Design System — Tipografia e Componentes Base

## Status

Ready for Review

## Story

**As a** desenvolvedor,
**I want** ter os componentes atômicos do design system implementados em React,
**so that** as próximas stories possam montar telas usando peças prontas e consistentes.

## Acceptance Criteria

1. Escala tipográfica implementada em `tokens.css`: H1–H3 + body + small conforme front-end-spec Seção 7.2
2. Componente `Button` com variantes `primary` e `ghost`, estados: hover, active, disabled, loading
3. Componente `ProgressBar` com prop `value` (0–100), animação 300ms ease-out, texto "X de 20"
4. Componente `Badge` com variante `locked` ("🔒 Bloqueado"), white-space: nowrap, flex-shrink: 0
5. Componente `Input` (type text/email) com estados: idle, focused, error; label + mensagem de erro inline
6. Componente `AnswerCard` (card de resposta do quiz): estados unselected, selected, hover
7. Todos os componentes usam tokens CSS — zero valores hardcoded
8. Componentes exportados de `src/components/index.js`

- Hover: brightness(1.15)

- Width: 100% mobile
- Loading: spinner CSS (border animation) inline left do texto
