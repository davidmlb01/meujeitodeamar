## User Story

**Como** membro da equipe UNLMTD ao enviar uma proposta para um cliente,
**Quero** ter um template de proposta comercial que transmita profissionalismo e identidade,
**Para que** a proposta feche antes mesmo do cliente ler o preço.

---

## Contexto

A UNLMTD trabalha com grandes clientes (Unilever, Google, Coca-Cola, Heineken). A proposta comercial é o documento que fecha negócios. Precisa ter a mesma força visual da apresentação institucional, mas com estrutura adequada para proposta.

---

## Acceptance Criteria

- [ ] Template em formato A4 vertical E formato digital 16:9
- [ ] Seções obrigatórias:
  - [ ] Capa (com nome do cliente e data)
  - [ ] Sobre a UNLMTD (quem somos — versão concisa)
  - [ ] O desafio (o problema do cliente)
  - [ ] Nossa abordagem (como a gente pensa)
  - [ ] A proposta (o que vamos fazer)
  - [ ] Equipe envolvida
  - [ ] Investimento (tabela de valores)
  - [ ] Cronograma
  - [ ] Próximos passos
  - [ ] Encerramento / contato
- [ ] Identidade visual consistente com a apresentação (5.7)
- [ ] Entregue em Figma e formato Word/Google Docs editável
- [ ] Guia de preenchimento para a equipe
- [ ] Documentado na página do brandbook

## Notas de Design

- A proposta deve ter a mesma energia da apresentação — não ser um documento Word genérico
- Seção de investimento precisa de layout digno do valor cobrado
- O logo/símbolo no cabeçalho/rodapé reforça a marca em cada página
- Versão A4 para impressão, 16:9 para envio digital/apresentação ao vivo
