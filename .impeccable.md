# .impeccable — Design Context

---

## Design Context — meujeitodeamar.com.br (Projeto Freud)

### Users
Mulheres 18-35, descobrindo autoconhecimento via TikTok/Instagram. Chegam com curiosidade e vulnerabilidade após 20 perguntas sobre padrões afetivos. O momento do resultado é altamente emocional: querem ser vistas, reconhecidas, compreendidas.

### Brand Personality
Íntima. Precisa. Transformadora.

### Emotional Goal
Revelação impactante. O resultado deve parecer que muda a forma como ela se enxerga. Tensão visual, contraste, peso. Não é um quiz de entretenimento, é um espelho.

### Aesthetic Direction
- Nude (#E8D9C1) como fundo calmo e sensorial
- Burgundy (#4B1D3F) como cor de poder e destaque
- Serif Cormorant Garamond para momentos de impacto emocional
- Jost para clareza e leitura fluida
- Sem cards genéricos, sem UI de SaaS. Tudo editorial, quase literário

### Design Principles
1. **O resultado é o herói.** Headline inconfundível, impossível de ignorar. Sem timidez tipográfica.
2. **Tensão visual gera desejo.** Conteúdo bloqueado deve parecer quase acessível (cards desfocados, não listas acinzentadas).
3. **Impacto dramático, não barulho.** Contraste de tamanho e peso, não de cores gritantes ou animações excessivas.
4. **Cada palavra tem peso.** O copy já é forte. O design deve amplificar, nunca competir.
5. **Mobile-first com presença.** Espaçamento generoso, tipografia que respira.

## Design Context

### Users
David Levy — founder solo gerenciando 4 projetos em paralelo (Freud, EasySite, Destaka, UNLMTD). Usa o kanban como painel de controle pessoal, não como ferramenta de time. Contexto: foco, clareza, velocidade de leitura. O trabalho é denso e a interface precisa reduzir carga cognitiva, não adicionar.

### Brand Personality
Pessoal, executivo, sem frescura. 3 palavras: **Foco. Controle. Clareza.**
Emocional: sensação de "tudo sob controle" ao abrir. Nenhuma ansiedade.

### Aesthetic Direction
- **Minimalista clean** — muito espaço, tipografia forte, zero decoração
- **Referência:** Linear app, Vercel dashboard, Raycast
- **Anti-referência:** Jira (poluído), Monday (colorido demais), Trello padrão (infantil)
- **Tema:** Dark mode exclusivo
- **Fonte:** Inter (Google Fonts)

### Pain Points Identificados
- Cards sem hierarquia visual — todos parecem iguais
- Header genérico sem personalidade
- Colunas indistinguíveis visualmente
- Falta de microanimações e transições fluidas

### Design Principles
1. **Espaço é conteúdo** — padding generoso, nada apertado
2. **Hierarquia pela tipografia** — tamanho e peso fazem o trabalho, não cor
3. **Cor com propósito** — cada projeto tem sua cor, usada com moderação (tag apenas)
4. **Zero ruído visual** — sem bordas desnecessárias, sem sombras pesadas
5. **Status óbvio em 2 segundos** — ao abrir, David sabe imediatamente o que está em andamento
6. **Colunas com identidade** — cada status tem uma assinatura visual clara (top accent bar)
7. **Microinterações** — hover, drag, drop com feedback visual fluido
